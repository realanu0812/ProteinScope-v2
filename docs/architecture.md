# Architecture

ProteinScope v2 uses a source-separated RAG architecture.

## Layers

1. Evidence Layer
   - Research papers
   - PubMed
   - Scientific articles

2. Community Layer
   - Reddit discussions
   - User experiences

3. Retrieval Layer
   - Dense retrieval
   - Hybrid search
   - Reranking

4. Generation Layer
   - Citation-backed answers
   - Evidence vs experience separation
